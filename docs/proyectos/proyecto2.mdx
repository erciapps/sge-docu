# Proyecto 2

## Enunciado
A partir de la temática asignada, finaliza (de forma **individual**) el módulo para que 
sea completamente funcional. El módulo debe tener:
* Todos los campos y modelos (entidades) propuestos.
* Incluir validaciones.
* Generar al menos 1 informe de 2 entidades.

## Entrega
* Subir a la tarea habilitada en Educamos:
  - Archivo zip del proyecto
  - PDF del manual básico de funcionamiento, indicando los pasos para crear o modificar, además de las validaciones que realiza, y como genera los informes PDF.
  - Archivo con el enlace a github de tu módulo.
  - PDFs de los informes generados.
* Realizar la entrega antes de la hora indicada, no esperes al último momento.
